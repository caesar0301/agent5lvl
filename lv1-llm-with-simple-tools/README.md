# Level 1

**旅行规划助理** 

个人旅行规划助理，针对用户消息中有关旅行的信息，利用大模型进行交互，并在结束时提问相关问题进行引导对
话。

使用步骤：
1. 登录Dify，创建一个新项目，并设置基础模型（示例中为Qwen）。
2. 导入DSL，开始使用。

**新闻助手** 

个人定制化的新闻助手，能够利用搜索引擎对用户消息中的关键词进行搜索，并通过大模型对搜索结果进行整理，以Markdown形式输出。

使用步骤：
1. 登录Dify，创建一个新项目，并设置基础模型（示例中为Qwen）。
2. 导入DSL。
3. 获取GoogleSearch API Key，并设置在Google搜索节点。
4. 开始使用。
